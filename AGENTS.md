# Agent instructions

- При добавлении нового экрана всегда создавай отдельную папку с `README.md`, описывающим фактический функционал экрана.
- При добавлении/изменении экранов обновляй `Screens.md`, указывая route и параметры открытия.
- Придерживайся MVVM подхода. Каждая фича (экран и его логика) в своей папке. Минимум переиспользования кода.
