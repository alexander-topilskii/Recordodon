# Recordodon

Диктофон: записывает аудио с утройства (В том числе с мобильных устройств, предварительно запрашивает пермишены на запись аудио)
- записывает в WAV- во время записи есть кнопка завершения, паузы, и маркера (флажок) 
- при клике на флажок - ставит маркер времени на дорожку (Который поддерживается oceanaudio) 
- в конце сохраняет файл и предлагает его "пошарить"- во время записи отображает гистограму аудио
