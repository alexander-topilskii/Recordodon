# Screens

Краткое описание экранов приложения и параметров их открытия.

| Экран | Route | Параметры | Описание |
| --- | --- | --- | --- |
| Главный список записей | `main` | Нет | Список аудиозаписей из папки сохранения по умолчанию, кнопка создания новой записи, меню с переходами в настройки и экран «О приложении». |
| Настройки | `settings` | Нет | Экран настроек с выбором папки сохранения и кнопкой возврата назад. |
| О приложении | `about` | Нет | Пустой экран информации о приложении с кнопкой возврата назад. |
| Рекордер | `recorder` | Нет | Экран записи с проверкой разрешения микрофона, управлением записью, гистограммой и маркерами WAV. |
