# Screens

Краткое описание экранов приложения и параметров их открытия.

| Экран | Route | Параметры | Описание |
| --- | --- | --- | --- |
| Главный список записей | `main` | Нет | Список аудиозаписей, кнопка создания новой записи, меню с переходами в настройки и экран «О приложении». |
| Настройки | `settings` | Нет | Пустой экран настроек с кнопкой возврата назад. |
| О приложении | `about` | Нет | Пустой экран информации о приложении с кнопкой возврата назад. |
| Рекордер | `recorder` | Нет | Заглушка экрана рекордера с кнопкой возврата назад. |
